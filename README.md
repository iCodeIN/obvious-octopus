# obvious octopus

## Getting Started

First, get the code on your system!  The simplest method is via git:
- `cd ~/`
- `git clone https://github.com/jorisschellekens/obvious-octopus.git`
- `cd obvious-octopus`
- `bash compile_script.sh`
