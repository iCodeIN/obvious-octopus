Detailed Description

The NGramLanguageDetector is based on calculating and comparing language profiles of N-gram frequencies. The system generates a language profile for the N-grams in particular language by using training data for the language in question and later uses these profiles to make its detection. Given a novel piece of text to be classified, the system computes the N-gram profile of this text (text profile) and compares the distance between this text profile and the language profiles for all the supported languages. The language profile with the minimal distance is considered to represent the detected language.
