Detailed Description

A heat map (or heatmap) is a graphical representation of data where the individual values contained in a matrix are represented as colors. The term Heatmap was originally coined and trademarked by software designer Cormac Kinney in 1991, to describe a 2D display depicting real time financial market information. Heat maps originated in 2D displays of the values in a data matrix. Larger values were represented by small dark gray or black squares (pixels) and smaller values by lighter squares.
