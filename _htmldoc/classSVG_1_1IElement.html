void SVG::IElement::rotate ( double  a)
inline

This transform definition specifies a rotation by a degrees about a given point. The rotate is about the origin of the current user coordinate system. The operation corresponds to the matrix \( \begin{pmatrix} cos(a) & -sin(a) & 0 \\ sin(a) & cos(a) & 0 \\ 0 & 0 & 1 \\ \end{pmatrix} \)
